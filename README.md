# ドラゴンゲームパーク

スマホ中心で遊べる Web ゲーム集です。  
最初のゲームとして、木製ボード風のカラハ式マンカラを実装しています。

## 技術構成

- React 19
- TypeScript
- Vite 8

## 主な機能

- ゲーム選択画面
- マンカラ設定画面
- マンカラ対局画面
- 人間 vs 人間
- 人間 vs CPU
- カラハ式マンカラルール（追加ターン・捕獲）
- 石が宙を飛ぶアニメーション
- スマホ・タブレット・PC 対応 UI

## 開発コマンド

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動（ブラウザで http://localhost:5173 を開く）
npm run dev

# 自動テストの実行
npm run test

# 本番用ビルド
npm run build
```
